プライバシーポリシー

基本方針

データクエリサービス（以下「当社」という）は、個人情報の重要性を認識し、個人情報保護法その他関連法令を遵守し、個人情報の適切な取扱いに努めております。

1. 個人情報の定義

本ポリシーにおける「個人情報」とは、生存する個人に関する情報であって、当該情報に含まれる氏名、連絡先その他の記述等により特定の個人を識別することができるもの（他の情報と容易に照合することができ、それにより特定の個人を識別することができることとなるものを含む）をいいます。

2. 個人情報の取得

当社は以下の方法により個人情報を収集いたします：

  • • 本サービスの利用登録時にご入力いただく情報
  • • お問い合わせ時の連絡先情報
  • • メールマガジン登録時の情報
  • • LINEその他SNSサービスを通じて取得する情報
  • • Cookieやアクセスログ等の利用情報

3. 取得する個人情報の項目

当社が収集する個人情報は以下の通りです：

  • • 氏名、ニックネーム
  • • メールアドレス
  • • 電話番号
  • • LINE IDその他SNSアカウント情報
  • • IPアドレス、ご利用環境情報
  • • サービス利用履歴、アクセスログ
  • • サービス提供に必要なその他の情報

4. 個人情報の利用目的

当社が収集した個人情報は、以下の目的で利用いたします：

  • ✓ 本プラットフォームの提供・運営
  • ✓ ユーザーへの連絡・回答
  • ✓ 分析レポートの配信
  • ✓ 分析によるプラットフォームの改善
  • ✓ 新機能やサービスに関する情報提供
  • ✓ 利用規約違反行為への対処
  • ✓ システムメンテナンス・障害対応
  • ✓ 統計データの作成（個人を識別できない形式で）

5. 第三者への個人情報の提供

当社は、以下の場合を除き、個人情報を第三者に提供いたしません：

  • • ご本人の同意がある場合
  • • 法令により要求される場合
  • • 人の生命、身体または財産の保護のために必要がある場合
  • • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合
  • • 国の機関若しくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合

6. 個人情報の委託

当社は、サービス提供のため以下の業務を外部委託する場合があります：

  • • システム開発・保守業務
  • • データ分析業務
  • • メール配信サービス
  • • カスタマーサポート運営

委託先に対しては、個人情報保護に関する契約を締結し、適切な管理を行います。

7. Cookieの利用

当社では、サービス改善のためCookieを利用しています：

  • • ユーザー利便性の向上
  • • アクセス解析
  • • 広告の最適化
  • • セキュリティの確保

ブラウザの設定によりCookieを無効にすることが可能ですが、その場合サービスの一部機能がご利用いただけなくなる可能性があります。

8. 個人情報の安全管理

個人情報の漏洩、滅失または毀損の防止その他の安全管理のため、以下の措置を講じています：

  • 🔒 アクセス権限の制限
  • 🔒 データの暗号化
  • 🔒 ファイアウォールの設置
  • 🔒 定期的なセキュリティ監査
  • 🔒 従業員への教育・研修

9. 個人情報の開示・訂正・削除

ご本人は、当社に対して以下の請求を行うことができます：

  • 📋 個人情報の開示
  • 📝 個人情報の訂正・追加・削除
  • ⏹️ 個人情報の利用停止・削除
  • 🚫 第三者提供の停止

ご請求いただいた場合、ご本人確認の上、合理的な期間内に対応いたします。

10. 個人データの保存期間

個人情報の保存期間は、利用目的の達成に必要な期間とし、法令に定めがある場合はその期間に従います。サービス終了後も、法的義務の履行や紛争解決のため、一定期間個人情報を保持する場合があります。

11. 未成年者の個人情報

当社では原則として20歳未満の方の個人情報は収集いたしません。未成年者がサービスをご利用される場合は、保護者の同意を得ていただく必要があります。

12. 本プライバシーポリシーの変更

当社は、法令の改正やサービス内容の変更等に応じて、本ポリシーを変更することがあります。重要な変更がある場合は、ウェブサイトでの告知等によりユーザーに通知いたします。
